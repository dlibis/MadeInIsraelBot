# MadeInIsraelBot

### Environment variables
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `MONDAY_API_TOKEN`
- `CORS_POLICY`

### API
- `GET /tables` - Tables + ID + Names + Views
    - ```
        {
            [
                {
                    "table_name": "name",
                    "table_id": "id",
                    "table_views": [
                        {
                            "view_name": "name",
                            "view_id": "id
                        },
                        ...
                    ]
                },
                ...
            ]
        }
        ```
- `GET /tables/[id]/rows` - Table row names
    -   ```
        {
            [
                {
                    "row_name": "name",
                    "row_id":, "id",
                },
                ...
            ]
        }
        ```
- `POST /message` - Template ID, arguments
